1
Gehe zu console.anthropic.com und erstelle einen kostenlosen Account.
2
Klicke links auf API Keys → Create Key. Kopiere den Key (beginnt mit sk-ant-).
3
Füge ihn unten ein und tippe auf Aktivieren. Fertig.
🔒 Dein Key wird ausschließlich lokal in diesem Browser gespeichert und nie an Server übertragen – nur direkt an die Anthropic API beim Scan.
